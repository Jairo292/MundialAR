# Mundial AR 2026

Sitio web informativo sobre el Mundial AR 2026.

## Tecnologías

- HTML5
- CSS3
- JavaScript

## Cómo usar

1. Clona el repositorio:
   ```sh
   git clone https://github.com/tuusuario/mundial-ar-2026.git
   ```
2. Abre el archivo `index.html` en tu navegador.

## Estructura del proyecto

- `index.html`: Página principal.
- `styles.css`: Estilos del sitio.
- `script.js`, `galeria.js`, `trivia.js`: Funcionalidad JS.
- `assets/`: Recursos multimedia.
- `tools/`: Herramientas y utilidades.
